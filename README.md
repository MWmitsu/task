# 📌 やること（Web版）— iPad・スマホ・どのブラウザでも

Windows常駐ウィジェットと**同じ感覚で使えるタスク管理**を、**iPad（Safari）やスマホ、どのPCのブラウザ**でも動くようにしたWeb版です。インストール不要・追加費用なし・データは端末の中だけ（ブラウザの保存領域）。

`index.html` 1ファイルだけで動きます。

## iPadで使う（おすすめ：ホーム画面に追加）
1. この `index.html` をiPadに用意する（どれかでOK）
   - **OneDrive/iCloud等に置く** → iPadのファイル/ブラウザから開く
   - **メールやAirDropで送る** → 受け取って開く
2. Safariで開いたら、**共有ボタン → 「ホーム画面に追加」**
3. ホーム画面のアイコンから、**アプリのように全画面**で使えます

> データはその端末のSafari内に保存されます（オフラインで完結）。アプリを消す/履歴消去すると消えるので、大事なときは「⇅ → JSON書き出し」でバックアップを。

## できること（Windows版の主要機能を網羅）
- タスク追加（改行で複数まとめて・`- ・ 1.` の記号は自動除去）
- 「**明日 ○○**」「**金曜 ○○**」で締切を自動セット
- 分類（すぐ／通常／いつでも）・フィルタタブ（すべて／今日／すぐ／いつでも）
- 締切が今日・期限切れは「今日」タブに集約（Windows版と同じ挙動）
- 📍優先度（低／中／高）・🏷タグ（タップでそのタグを検索）・📐見積もり・📝備考
- 完了（取り消し線）・今日の完了数＆連続日数（🔥）
- 検索・一時メモ・並べ替え・複製・完了を隠す/消す

## クラウド同期（複数端末で自動・麻雀アプリと同じ方式）
右上「⇅」→「**🔑 Googleでログイン**」すると、**iPad・PCのブラウザなど複数端末で自動同期**されます（麻雀スコアアプリと同じ Firebase 方式）。
- ログイン中は、編集すると自動でクラウドへ保存され、他端末にもすぐ反映（`onSnapshot`）
- 端末とクラウドの両方にデータがある初回は「どちらを使うか」を確認
- オフラインや未ログインでもローカルだけで普通に使えます（同期だけ止まる）
- ※ `file://`（ファイル直接開き）ではGoogleログイン不可 → **ホスティング版URL**（後述）で使ってください

### Windows版とのデータのやり取り（ファイル経由）
クラウド同期とは別に、**Windows常駐版との橋渡し**はファイル（CSV/JSON）で行います。

| やりたいこと | 手順 |
|---|---|
| **Windows版 → iPad** | Windows版でCSV書き出し（または `widget-data.json` をそのまま）→ クラウド/メールでiPadへ → Web版で「📥 取り込み」 |
| **iPad → Windows版** | Web版で「📤 CSV書き出し」→ そのCSVをWindows版のトレイ「CSVから取り込み」で読込 |
| **バックアップ** | Web版「📤 JSON書き出し」で全体を保存 |

- CSVの形式（列・タグの区切り・文字コードUTF-8 BOM）はWindows版と**互換**にしてあります
- 取り込みは「**本文が同じタスクは重複として除外**」して追加します

## 仕組み（技術メモ）
- 単一HTMLファイル（HTML+CSS+JS）。フレームワーク不要。`localStorage` に保存（キー `taskweb-data-v1`）
- CSVは Windows版 `Export-Tasks-Csv` と同じ列順（text,done,cat,priority,due,estimate_min,tags,note）・タグは `|` 区切り・先頭にBOM
- JSON取り込みは Web版の書き出しと Windows版 `widget-data.json` の両方を受け付け（分類はキー/ラベルどちらも解釈）
- ローカル確認用サーバ：`powershell -ExecutionPolicy Bypass -File serve.ps1 -Port 8190`（iPadで使うだけなら不要。ファイルを開くだけで動きます）
